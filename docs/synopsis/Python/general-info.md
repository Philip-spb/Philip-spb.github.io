# Общая информация

## Базовые типы данных

| тип      | описание                     |
| -------- | ---------------------------- |
| int      | целые числа *123*            |
| float    | вещественные числа *123.0*   |
| str      | строки                       |
| bool     | булевы переменные            |
| NoneType | None type                    |
| complex  | комплексные числа *2.0 + 3j* |
| bytes    | байтовая строка              |
